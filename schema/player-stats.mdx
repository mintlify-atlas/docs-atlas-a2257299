## Overview

Player statistics are organized across multiple CSV files, each focusing on different aspects of performance. All player datasets share common identifier fields and can be joined using `nba_id` (or `PLAYER_ID`) and `year`.

## Core Player Files

### scoring.csv

Basic scoring efficiency and volume metrics.

**File**: `scoring.csv` / `scoring_ps.csv`

<ResponseField name="Player" type="string" required>
  Player full name
</ResponseField>

<ResponseField name="nba_id" type="integer" required>
  Unique player identifier for joining with other datasets
</ResponseField>

<ResponseField name="year" type="integer" required>
  Season year (e.g., 2014, 2015)
</ResponseField>

<ResponseField name="Tm" type="string">
  Team abbreviation (e.g., "LAL", "GSW", "TOT" for players traded mid-season)
</ResponseField>

<ResponseField name="G" type="integer">
  Games played
</ResponseField>

<ResponseField name="MP" type="float">
  Total minutes played for the season
</ResponseField>

<ResponseField name="PTS" type="float">
  Total points scored
</ResponseField>

<ResponseField name="TS%" type="float">
  True shooting percentage: `PTS / (2 * (FGA + 0.44 * FTA))`
  
  Accounts for the value of 3-pointers and free throws.
</ResponseField>

**Example**:
```csv
Player,TS%,PTS,MP,Tm,G,year,nba_id
Zaid Abdul-Aziz,51.5,15.9,2459.0,HOU,79,1974,76002.0
```

### player_shooting.csv

Detailed shooting metrics broken down by shot coverage and distance.

**File**: `player_shooting.csv` / `player_shooting_p.csv`

<ResponseField name="PLAYER_ID" type="integer" required>
  Unique player identifier (equivalent to `nba_id`)
</ResponseField>

  - `very_tight`: Defender 0-2 feet away
</ResponseField>

<ResponseField name="TEAM" type="string">
  Team abbreviation
</ResponseField>

<ResponseField name="AGE" type="float">
  Player age during the season
</ResponseField>

<ResponseField name="GP" type="integer">
  Games played
</ResponseField>

<ResponseField name="G" type="integer">
  Games started
</ResponseField>

<ResponseField name="FREQ%" type="float">

</ResponseField>

<ResponseField name="FGM" type="integer">
  Field goals made
</ResponseField>

<ResponseField name="FGA" type="integer">
  Field goals attempted
</ResponseField>

<ResponseField name="FG%" type="float">
  Field goal percentage (0-100 scale)
</ResponseField>

<ResponseField name="EFG%" type="float">
  Effective field goal percentage: `(FGM + 0.5 * 3PM) / FGA`
</ResponseField>

<ResponseField name="2FG FREQ%" type="float">

</ResponseField>

<ResponseField name="2FGM" type="integer">
  2-point field goals made
</ResponseField>

<ResponseField name="2FGA" type="integer">
  2-point field goals attempted
</ResponseField>

<ResponseField name="2FG%" type="float">

</ResponseField>

<ResponseField name="3FG FREQ%" type="float">

</ResponseField>

<ResponseField name="3PM" type="integer">
  3-pointers made
</ResponseField>

<ResponseField name="3PA" type="integer">
  3-pointers attempted
</ResponseField>

<ResponseField name="3P%" type="float">
  3-point percentage
</ResponseField>

## Usage Examples

### Finding Elite Catch-and-Shoot Players

```python
import pandas as pd

# Load player shooting data
shooting = pd.read_csv('player_shooting.csv')

# Filter for wide open 3-point attempts
wide_open_3s = shooting[
    (shooting['shot_type'] == 'wide_open') &
    (shooting['3PA'] >= 100)  # Minimum attempts
].sort_values('3P%', ascending=False)

print(wide_open_3s[['PLAYER', 'TEAM', '3PM', '3PA', '3P%', 'year']].head(10))
```

### Analyzing Hustle Impact

```python
# Load hustle data
hustle = pd.read_csv('hustle.csv')

# Players who excel at deflections per minute
hustle['DEFL_PER_MIN'] = hustle['DEFLECTIONS'] / hustle['MIN']

top_deflectors = hustle[
    hustle['MIN'] >= 1000
].sort_values('DEFL_PER_MIN', ascending=False)

print(top_deflectors[['PLAYER_NAME', 'TEAM_ABBREVIATION', 'DEFLECTIONS', 'DEFL_PER_MIN']].head(10))
```

### Join Multiple Player Datasets

```python
# Combine scoring efficiency with tracking data
scoring = pd.read_csv('scoring.csv')
tracking = pd.read_csv('tracking.csv')

# Filter tracking for drives only
drives = tracking[tracking['type'] == 'drives']

# Merge on player ID and year
merged = pd.merge(
    scoring,
    drives,
    left_on=['nba_id', 'year'],
    right_on=['PLAYER_ID', 'year'],
    how='inner'
)

print(merged[['Player', 'TS%', 'PTS', 'DRIVES', 'FG%']].head())
```
